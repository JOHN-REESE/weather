# 天气预报查询工具

基于经纬度查询未来7天天气预报的Python脚本。

## 功能特性

- 🌍 支持全球任意地点的经纬度查询
- 📅 提供未来7天详细天气预报
- 🌡️ 显示每日最高/最低温度
- 🌧️ 包含降水量、风速等详细信息
- 🌅 显示日出日落时间
- 🈳 完全免费，无需API密钥
- 🇨🇳 中文界面和天气描述

## 使用方法

### 1. 命令行参数方式
```bash
python3 weather_forecast.py [纬度] [经度]

# 例如查询北京天气
python3 weather_forecast.py 39.9042 116.4074

# 例如查询福州天气
python3 weather_forecast.py 26.0745 119.2965
```

### 2. 交互输入方式
```bash
python3 weather_forecast.py
# 然后根据提示输入纬度和经度
```
