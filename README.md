# 天气预报查询工具

基于城市名称或经纬度查询未来7天天气预报的Python脚本。

## 功能特性

- 🌍 支持全球任意地点的城市名称或经纬度查询
- 🏙️ 智能地理编码，支持中英文城市名称
- 📅 提供未来7天详细天气预报
- 🌡️ 显示每日最高/最低温度
- 🌧️ 包含降水量、风速等详细信息
- 🌅 显示日出日落时间
- 🔄 交互模式支持连续查询多个地点
- 🈳 完全免费，无需API密钥
- 🇨🇳 中文界面和天气描述

## 使用方法

### 1. 命令行参数方式
```bash
python3 weather_forecast.py [城市名称或坐标]

# 例如查询北京天气
python3 weather_forecast.py 北京

# 例如查询福州天气
python3 weather_forecast.py 26.0745,119.2965

# 例如查询纽约天气
python3 weather_forecast.py New York
```

### 2. 交互模式（推荐）
```bash
python3 weather_forecast.py
```
启动后进入交互模式：
- 输入城市名称或坐标进行查询
- 查询完成后按回车键继续查询下一个地点
- 输入 `quit`、`exit` 或 `q` 退出程序
- 支持连续查询多个地点，无需重新启动程序

## 数据来源

本工具使用以下免费API服务：
- **天气数据**: [Open-Meteo API](https://open-meteo.com) - 提供全球天气预报数据
- **地理编码**: [OpenStreetMap Nominatim](https://nominatim.openstreetmap.org) - 将城市名称转换为经纬度坐标
